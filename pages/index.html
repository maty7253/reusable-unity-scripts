Accelerate Your Unity Development

A curated collection of C# scripts designed to speed up your Unity 6 projects with ready-to-use components for both 2D and 3D game development.

Components specifically designed for 2D game development

  • Player following systems
  • Object pooling
  • Level transitions

Available 2D Scripts

CameraShaker.cs

Add screen shake effects to enhance game feel and impact

followPlayer.cs

Smooth camera following system for player movement

hueCycler.cs

Cycle through hue values for dynamic color effects

LevelTeleporter.cs

Handle scene transitions and level teleportation

ObjectPool.cs

Efficient object pooling system for performance optimization

rotateOnMove.cs

Rotate objects based on movement direction

How to Use

1

Clone or Download

Clone the repository or download individual scripts from the GitHub repository

git clone https://github.com/pilot2254/reusable-unity-6-scripts.git
2

Import to Unity

Copy the desired scripts to your Unity project's Assets folder

3

Attach and Configure

Attach scripts to GameObjects and configure parameters in the Inspector

Contribute

Help grow this collection by contributing your own Unity scripts. Check out the contributing guidelines and submit a pull request.
